# シーンデータコピー

別のシーンファイルから、アセット・設定データを現在のシーンに選択的にコピーできるパネルです。

---

## 何ができるか

過去に作ったシーンの「このアセットだけ」「このライト設定だけ」を今のシーンに持ってきたい場合に使います。

コピー対象として選択できるデータ:

| タブ | 内容 |
|------|------|
| Assets | アセット（カメラ・ライト・アバター等） |
| Blueprints | ノードのBP（ブループリント） |
| LightOp Groups | ライト操作のグループ設定 |
| LaserOp Groups | レーザー操作のグループ設定 |
| CameraOp Presets | カメラ操作のプリセット |

---

## 使い方

1. パネルを開く
2. **「ソースsceneファイルを選択」** で元にするシーンJSONを指定する
3. タブを切り替えてコピーしたいデータを選ぶ（チェックボックスで選択）
4. **「コピー実行」** ボタンを押す

---

## 注意事項

- 同名のデータがあっても**上書きはされません**（別データとして追加されます）
- IDが重複した場合は自動で新しいIDが採番されます
- コピー操作はUndo（Ctrl+Z）で元に戻せます

---

    - テンプレートシーンを作っておいて、各楽曲シーンにコピーする
