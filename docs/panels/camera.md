# カメラ操作

カメラアセットを管理し、映像出力・自動カメラワークを制御するパネルです。

---

## カメラアセット

複数のカメラをアセットとして登録し、用途ごとに使い分けられます。

### カメラの追加

1. カメラ操作パネル上部の **「追加」** ボタンをクリック
2. カメラ名を入力して確定

---

## Spout2送信

各カメラの映像をSpout2でOBSなどの外部ソフトに送信できます。

| 設定項目   | 説明                                                 |
| ---------- | ---------------------------------------------------- |
| Spout2送信 | ON/OFFを切り替え                                     |
| Spout名    | 自動で `AvatarSystem Camera + カメラ名` が設定される |

!!! tip
OBSでの受信には [obs-spout2-plugin](https://github.com/Off-World-Live/obs-spout2-plugin) が必要です。

---

## 自動カメラワーク

指定したオブジェクトを自動で追跡するカメラワーク機能です。

| 設定項目         | 説明                                                 |
| ---------------- | ---------------------------------------------------- |
| 対象カメラ       | 自動カメラワークを適用するカメラを選択               |
| 追跡対象         | 追跡するオブジェクト（アバター・アイテムなど）を選択 |
| 関節ターゲット   | アバターを選択した場合、追跡する関節を選択           |
| カメラワーク種類 | 使用するカメラワークのパターンを選択                 |

---

## ロック機能

カメラロックをONにすると、Unity内でのカメラ操作が無効になります。  
ライブ中の誤操作防止に使用してください。

---

## スクリーンショット

1. ドロップダウンリストから撮影するカメラを選択
2. **「撮影」** ボタンを押す
3. `Datas/Screenshots/` フォルダに保存される

解像度を変更して撮影することも可能です。
