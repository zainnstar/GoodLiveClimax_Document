# PixelFixtureライト

複数の発光ドットを1つのフィクスチャーとして描画する特殊なライトアセットです。  
LEDパネル・ドットスクリーンのような演出に使用します。

---

## 概要

PixelFixtureLightは、1枚のクアッド（平面）上に**グリッド状の発光ドットをまとめて描画**します。

- 1フィクスチャーが1つのRendererとして動作するため、多数のドットを軽量に描画できます
- 動画やテクスチャを入力として、各ドットにセル単位の色を割り当てられます

---

## パラメータ

| パラメータ | 説明 |
|-----------|------|
| グリッド数 (X/Y) | 横・縦のドット数 |
| ドットサイズ | 1ドットの描画サイズ |
| 発光強度 (Emission) | ドットの明るさ |
| 動画入力 | 入力テクスチャ・動画ソースの割り当て |

---

## 動画・テクスチャ入力

RenderTextureを経由して、動画やカメラ映像をPixelFixtureに入力できます。  
入力された映像はセル単位の色に変換されて各ドットに反映されます。

---

## PixelLightとの違い

| | PixelFixtureLight | PixelLight |
|-|-------------------|------------|
| 描画方式 | 1フィクスチャー = 1Renderer | 1灯 = 1インスタンス |
| 向いている用途 | ドット多数・動画入力 | 少数の個別制御 |
| 負荷 | 軽い | ドット数に比例 |
