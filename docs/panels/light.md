# ライト操作

ライトをグループ単位で管理し、ライブ会場のようなライティング演出を実現するパネルです。
プリセットとしてライトの動きなどを設定しておき、必要に応じて呼び出します。

---

## グループ管理

ボリュームライトをグループに分類して一括制御できます。

### グループの作成

ライトをグループに割り当てます。

### グループへのライト登録

---

## グループ単位の制御

| 制御項目 | 説明                                    |
| -------- | --------------------------------------- |
| 色       | グループ全体のライトカラーを変更        |
| 強さ     | グループ全体の光量を変更（0.0〜最大値） |
| ON/OFF   | グループ全体を一括でON/OFF              |

!!! info
グループ内のライト位置を変更すると、アセットリスト側にも反映されます。

---

## OSC連携

外部のChataigneやコントローラから、OSC経由でライトを制御できます。

## シャドウ設定

各ライトアセットでキャストシャドウ・レシーブシャドウの設定が可能です。

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
