# ライト操作

ライトをグループ単位で管理し、ライブ会場のようなライティング演出を実現するパネルです。

---

## ライトアセット

### ライトの追加

1. ライト操作パネルの **「追加」** ボタンをクリック
2. ライトタイプを選択（ポイントライト・スポットライト等）
3. シーンにライトが配置される

---

## グループ管理

ライトをグループに分類して一括制御できます。

### グループの作成

テーブル形式でライトをグループに割り当てます。  
エクセルの表のように行・列でレイアウトを管理できます。

### グループへのライト登録

1. グループテーブルの対象セルを選択
2. アセットリストからライトをドラッグ＆ドロップして登録

---

## グループ単位の制御

| 制御項目 | 説明                                    |
| -------- | --------------------------------------- |
| 色       | グループ全体のライトカラーを変更        |
| 強さ     | グループ全体の光量を変更（0.0〜最大値） |
| ON/OFF   | グループ全体を一括でON/OFF              |

!!! info
グループ内のライト位置を変更すると、アセットリスト側にも反映されます。

---

## OSC連携

外部のChataigneやコントローラから、OSC経由でライトを制御できます。

## シャドウ設定

各ライトアセットでキャストシャドウ・レシーブシャドウの設定が可能です。

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
