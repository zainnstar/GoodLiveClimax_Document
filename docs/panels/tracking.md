# トラッキング

MediaPipeを使ったフルボディトラッキングの設定パネルです。

---

## 概要

GoodLiveClimaxはMediaPipe Unity Pluginを使用して、カメラ映像から以下のトラッキングデータを取得します。

| トラッキング対象 | 説明                                           |
| ---------------- | ---------------------------------------------- |
| 顔               | パーフェクトシンク対応（ブレンドシェイプ制御） |
| 手               | 指の動きをVMMで制御                            |
| 体               | 上半身ポーズ推定                               |
| 足               | 下半身ポーズ推定                               |

---

## セットアップ

### カメラの選択

1. トラッキングパネルの **「カメラ」** ドロップダウンから使用するWebカメラを選択
2. **「開始」** ボタンでトラッキングを開始

### 解像度・フレームレート設定

| 推奨設定       | 値      |
| -------------- | ------- |
| 解像度         | 640×480 |
| フレームレート | 30fps   |

高解像度・高フレームレートは精度が上がりますが、PC負荷も増加します。

---

## 顔トラッキング（パーフェクトシンク）

VRM 0.xのパーフェクトシンク対応アバターであれば、表情をリアルタイムに制御できます。

!!! warning "対応アバターについて"
パーフェクトシンクには対応ブレンドシェイプが必要です。  
 VRoidで書き出したアバターは標準では非対応のため、別途設定が必要です。

---

## 手トラッキング

MediaPipe Hand Landmarksをベースに、VMM（Virtually Model Manipulator）で手指の動きをアバターに反映します。

---

## OSC出力

トラッキングデータをOSC経由で外部ソフトに送信できます。
