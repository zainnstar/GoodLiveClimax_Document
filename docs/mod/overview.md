# MODとは

GoodLiveClimaxでは、ユーザーが独自に作成した**MOD（追加コンテンツ）**を読み込むことができます。  
MODを使うと、アバター・ステージ・小道具などを自分でカスタマイズして追加できます。

---

## MODでできること

- 独自の3Dモデル（ステージ・小道具）をアイテムとして追加
- カスタム背景・シーンの追加
- 独自エフェクト・パーティクルの追加

---

## MODファイル形式

MODは `.cmxmod` 形式の単一ファイルとして配布されます。

---

## MODの導入方法

1. `.cmxmod` ファイルをダウンロードする
2. 以下のフォルダに配置する

   | 種類                   | 配置先フォルダ               |
   | ---------------------- | ---------------------------- |
   | ワールド・ステージ     | `StreamingAssets/Worlds/`    |
   | アイテム・小道具       | `StreamingAssets/Items/`     |
   | アバター・キャラクター | `StreamingAssets/Avatars/`   |
   | パーティクル           | `StreamingAssets/Particles/` |

3. GoodLiveClimaxを起動（または再起動）する
4. アセットリストに新しいアイテムが表示される

---

## MODを自作したい方へ

自作MODの作り方は [MOD作成手順](creating-mod.md) を参照してください。  
作成に必要なSDKは [SDKダウンロード](sdk.md) から入手できます。
