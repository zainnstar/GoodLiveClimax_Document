# サイリウム（アセット）

コンサートで使われるケミカルライト（サイリウム）を再現するアセットです。ライブ演出のクラップ・振りなどに使用します。

---

## サイリウムの追加

1. 左サイドバーの **「サイリウム」** タブを開く
2. **「追加」** ボタンをクリック

---

## パラメータ

| パラメータ | 説明                     |
| ---------- | ------------------------ |
| カラー     | サイリウムの発光色       |
| 本数       | 同時に表示する本数       |
| 配置範囲   | サイリウムが分布する範囲 |

---

## アニメーション

サイリウムは振りに対応したアニメーションが再生されます。

- 目標位置への到達タイミングは同じに保ちつつ、到達までの動きにランダム性があります
- BPMに同期させることも可能です

---

## OSC連携

BPMなどの情報をOSC経由で受け取り、リアルタイムにサイリウムの動きを同期させられます。
