# スクリーン（アセット）

映像を投影するスクリーンアセットです。動画・静止画・カメラ映像をスクリーン面に表示できます。

---

## スクリーンの追加

1. 左サイドバーの **「スクリーン」** タブを開く
2. **「追加」** ボタンをクリック

---

## 表示コンテンツの設定

| コンテンツ種類 | 説明                                         |
| -------------- | -------------------------------------------- |
| 静止画         | PNGやJPGファイルをスクリーンに表示           |
| 動画           | 動画ファイルをループ再生                     |
| カメラ映像     | GoodLiveCLimax内のカメラアセットの映像を表示 |

---

## スクリーンエフェクト

ドットスクリーン（LEDビジョン風）などのエフェクトを追加できます。

---

## シャドウ設定

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
