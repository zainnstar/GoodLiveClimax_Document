# 3Dテキスト（アセット）

シーン内に3Dテキストを配置するアセットです。歌詞テロップ・タイトル表示などに使用します。

---

## 3Dテキストの追加

1. 左サイドバーの **「3Dテキスト」** タブを開く
2. **「追加」** ボタンをクリック

---

## パラメータ

| パラメータ           | デフォルト | 説明                   |
| -------------------- | ---------- | ---------------------- |
| テキスト内容         | (空)       | 表示するテキスト       |
| フォントサイズ       | 8          | テキストの大きさ       |
| カラー               | 白         | テキストカラー         |
| フォント             | -          | 使用するフォント       |
| 位置・回転・スケール | -          | トランスフォームで調整 |

---

## シャドウ設定

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | OFF        |
| Receive Shadow | ON         |

!!! info
3Dテキストは通常、影を落とさない設定がデフォルトです。
