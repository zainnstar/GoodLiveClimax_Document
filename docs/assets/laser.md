# レーザービーム（アセット）

ステージ演出用のレーザービームエフェクトアセットです。

---

## レーザービームの追加

1. 左サイドバーの **「レーザー」** タブを開く
2. **「追加」** ボタンをクリック

---

## パラメータ一覧

### 基本設定

| パラメータ | 値域 | 説明 |
|-----------|------|------|
| カラー（始点） | RGB | レーザー開始点の色 |
| カラー（終点） | RGB | レーザー終端の色 |
| 強さ (Intensity) | 0〜300 | レーザーの輝度 |
| 本数 (Count) | 1〜64 | 同時に表示するレーザーの本数 |
| 幅 (Width) | 0.0001〜0.1 | 1本あたりの太さ |

### 形状設定

| パラメータ | 値域 | 説明 |
|-----------|------|------|
| 扇角度 (Fan Angle) | 90〜360° | レーザーが広がる角度 |
| 開き (Open) | -0.05〜2.0 | 広がり量 |
| 長さ (Length) | 0.01〜3.0 m | レーザーの長さ |

### フェード設定

| パラメータ | 値域 | 説明 |
|-----------|------|------|
| 先端フェード強度 | 0〜1 | 先端が消えるグラデーション強さ |
| 先端フェードパワー | 0.1〜10 | フェードカーブの傾き |
| 縁フェード強度 | 0〜1 | 両端が消えるグラデーション強さ |

### ノイズ設定

| パラメータ | 値域 | 説明 |
|-----------|------|------|
| ノイズタイプ | 0〜2 | ノイズ形状の種類 |
| ノイズスケール | 0〜50 | ノイズの大きさ |
| ノイズ速度 | 0〜50 | ノイズの動く速さ |
| ノイズ強度 | 0〜1 | ノイズによる揺らぎの量 |

### 点滅・パルス

| パラメータ | 説明 |
|-----------|------|
| パルス ON/OFF | 規則的な点滅 |
| パルス速度 | 点滅の速さ(0.1〜5) |
| パルス最小強度 | 暗くなった時の最小輝度(0〜1) |
| フリッカー ON/OFF | 不規則なちらつき |
| フリッカー速度 | ちらつきの速さ(1〜120) |
