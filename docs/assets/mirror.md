# ミラー（アセット）

シーン内に鏡を配置するアセットです。反射表現によるリアルなステージ演出に使用します。

---

## ミラーの追加

1. 左サイドバーの **「ミラー」** タブを開く
2. **「追加」** ボタンをクリック
3. シーンにミラーが配置される

---

## パラメータ

| パラメータ | 説明                   |
| ---------- | ---------------------- |
| サイズ     | ミラーの幅・高さ       |
| 位置・回転 | トランスフォームで調整 |

---

## MSAA設定

ミラーのエッジ品質はMSAA（マルチサンプルアンチエイリアシング）設定で調整できます。

---

## シャドウ設定

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
