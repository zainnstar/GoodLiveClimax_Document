# アイテム（アセット）

シーンに3Dオブジェクトを配置するアセットです。小道具・ステージ装飾などに使用します。

---

## アイテムの追加

1. 左サイドバーの **「アイテム」** タブを開く
2. **「追加」** ボタンからファイルを選択

対応フォーマット: **glTF (.glb / .gltf)** など

---

## 共通プロパティ

| 設定             | 説明                               |
| ---------------- | ---------------------------------- |
| トランスフォーム | 位置・回転・スケールの調整         |
| マテリアル       | シェーダーパラメータをUIで直接変更 |
| Cast Shadow      | ON（デフォルト）                   |
| Receive Shadow   | ON（デフォルト）                   |

---

## アニメーション

アイテムに付属のアニメーション（`.anim`）を再生できます。  
`StreamingAssets/` フォルダに配置した `.anim` ファイルがドロップダウンリストに表示されます。
