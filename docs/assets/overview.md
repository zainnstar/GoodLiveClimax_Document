# アセット一覧

GoodLiveClimaxでは、演出要素をすべて「アセット」として管理します。  
アセットは左サイドバーのリストに一覧表示され、追加・削除・名前変更ができます。

---

## アセットの種類

| アセット | 用途 |
|---------|------|
| [アバター](avatar.md) | VRMアバターの表示・トラッキング制御 |
| [カメラ](camera.md) | カメラアングル管理・Spout2映像出力 |
| [ライト](light.md) | ダイレクショナル・スポット・ポイントライト |
| [レーザービーム](laser.md) | レーザーエフェクト描画 |
| [アイテム](item.md) | 3Dオブジェクト・小道具の配置 |
| [パーティクル](particle.md) | パーティクルエフェクト再生 |

---

## 共通操作

### アセットの追加

各タブの **「追加」** ボタン、またはドラッグ＆ドロップでシーンに配置できます。

### 名前変更

アセットリスト上のアセットを**右クリック → 「名前を変更」**。  
内部のIDは変わらないため、プリセットやノードでの参照が切れません。

### 共通プロパティ

すべてのアセットが持つ共通設定:

=== "トランスフォーム"
    | 項目 | 説明 |
    |------|------|
    | 位置 (Position) | X / Y / Z 座標 |
    | 回転 (Rotation) | X / Y / Z オイラー角 |
    | スケール (Scale) | X / Y / Z 倍率 |

=== "シャドウ設定"
    | 項目 | デフォルト |
    |------|-----------|
    | Cast Shadow（影を落とす） | ON |
    | Receive Shadow（影を受ける） | ON |

=== "マテリアル"
    アタッチされているマテリアルのシェーダーパラメータをUI上で直接調整できます。

---

## アセットIDについて

すべてのアセットは固有の**AssetID (GUID)**で管理されています。  
名前を変更してもIDは変わらないため、シーンデータの参照やノードの接続が保持されます。
