# アバター

VRMアバターを読み込み、トラッキングデータに連動して動かすアセットです。

---

## 対応フォーマット

- **VRM 0.x** 形式（VRM 1.xは非対応）
- パーフェクトシンク対応アバター推奨（表情を細かく制御する場合）

---

## アバターの追加

1. 左サイドバーの **「アバター」** タブを開く
2. **「追加」** ボタンをクリック
3. VRMファイルを選択して読み込む

!!! tip "VRMの配置場所"
VRMファイルはアプリ内の任意の場所から読み込めますが、  
 `Datas/Avatars/` フォルダに入れておくと管理がしやすくなります。

---

## トラッキング連携

[トラッキングパネル](../panels/tracking.md) でWebカメラのトラッキングを有効にすると、アバターの表情・手・体が自動で動きます。

| トラッキング種別         | 制御対象                  |
| ------------------------ | ------------------------- |
| 顔（パーフェクトシンク） | 表情ブレンドシェイプ 52種 |
| 手                       | 指の関節ボーン            |
| 体                       | 上半身・下半身ポーズ      |

---

## AddShader（シェーダー追加機能）

アバターにアドオンシェーダーを後付けできます。  
アバターのマテリアルを変更せずにエフェクトを追加できるのが特徴です。

現在対応しているエフェクト:

| エフェクト   | 説明                                                               |
| ------------ | ------------------------------------------------------------------ |
| アウトライン | Quick Outline ベースの後付けアウトライン。種類・色・太さを調整可能 |

---

## シャドウ設定

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
