# ライト（アセット）

シーンに配置するライトのアセットです。複数種類のライトをグループ管理して演出できます。

---

## ライトの種類

| 種類                     | 用途                       |
| ------------------------ | -------------------------- |
| ポイントライト           | 全方向に光を放射する点光源 |
| スポットライト           | コーン形状に光を照射       |
| ディレクショナルライト   | 太陽光のような平行光源     |
| ボリューメトリックライト | 光の筋（ゴッドレイ）を表現 |

---

## ライトの追加

1. 左サイドバーの **「ライト」** タブを開く
2. **「追加」** ボタンからライトタイプを選択
3. シーンにライトが配置される

---

## グループ管理

ライト操作パネルでライトをグループに分類し、グループ単位で色・強さをコントロールできます。  
詳細は [ライト操作パネル](../panels/light.md) を参照してください。

---

## ボリューメトリックライト

ライトビームの「光の筋」を描画する機能です。

| パラメータ                | 説明                        |
| ------------------------- | --------------------------- |
| ボリューメトリック ON/OFF | 光の筋の表示切り替え        |
| 光量                      | 筋の明るさ                  |
| ダウンサンプル            | GPU負荷調整（高いほど軽い） |
| 最大レイ長                | 光の筋が届く最大距離        |

!!! tip "負荷について"
ボリューメトリック機能をOFFにすると、カメラ側のレンダラーも自動でOFFになり負荷を削減できます。

---

## シャドウ設定

| 設定           | デフォルト |
| -------------- | ---------- |
| Cast Shadow    | ON         |
| Receive Shadow | ON         |
