# パーティクル（アセット）

Unityのパーティクルシステムを利用した演出エフェクトアセットです。

---

## パーティクルの追加

1. 左サイドバーの **「パーティクル」** タブを開く
2. **「追加」** ボタンをクリック
3. プレハブリストから使用するパーティクルを選択

!!! info "プレハブについて"
    パーティクルはプレハブ（`ParticleSystemAsset`）として管理されています。  
    Asset Type Id に `particleSystem` が設定されたプレハブが対象です。

---

## 操作

| 操作 | 説明 |
|------|------|
| 再生 | パーティクルを開始 |
| 一時停止 | 現在の状態で停止 |
| 再起動 | 最初から再生しなおす |

---

## トランスフォーム

パーティクルの発生位置・向きはトランスフォームで調整します。
