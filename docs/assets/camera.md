# カメラ（アセット）

シーン内のカメラを管理するアセットです。複数のカメラを登録して映像を切り替えられます。

---

## カメラの追加

1. 左サイドバーの **「カメラ」** タブを開く
2. **「追加」** ボタンをクリック

---

## Spout2送信

各カメラの映像をリアルタイムでOBSなどの外部ソフトに送信できます。

| 設定       | 説明                                        |
| ---------- | ------------------------------------------- |
| Spout2送信 | ON/OFFを切り替え                            |
| Spout名    | `AvatarSystem Camera + カメラ名` で自動設定 |

!!! info "OBSでの受信方法"
OBSで映像を受け取るには [obs-spout2-plugin](https://github.com/Off-World-Live/obs-spout2-plugin) が必要です。  
 OBSのソース追加から「Spout2 Capture」を選んでSpout名を合わせてください。

---

## カメラワーク

カメラ操作パネルと組み合わせることで、自動カメラワークが使えます。  
詳細は [カメラ操作パネル](../panels/camera.md) を参照してください。

---

## スクリーンショット

このカメラの映像をそのままPNG形式で保存できます。

1. カメラ操作パネルでカメラを選択
2. 解像度を設定（任意）
3. **「撮影」** ボタンを押す
4. `Datas/Screenshots/` に保存される

---

## ボリューメトリックライト対応

カメラアセットには **VolumetricLightRenderer** コンポーネントが組み込まれています。  
ボリューメトリックライトアセットの機能をOFFにしている場合、レンダラーも自動でOFFになります（GPU負荷軽減）。
