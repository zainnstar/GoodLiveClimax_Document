# GoodLiveClimax

**個人VTuber向けライブ演出ソフト**

GoodLiveClimaxは、個人VTuberが**一人でライブを成立させる**ことを目的として作られた演出エンジンです。  
Unityをバックエンドに、カメラ・ライト・レーザー・トラッキングなどの演出要素をコントロールできます。

---

## 主な特徴

- **ワンオペ前提の設計** — 当日の手動操作を最小限に抑えた自動演出
- **タイムライン連動** — Chataigneシーケンスと連携した再現性の高い演出
- **オーディオリアクティブ** — 音楽に自動で反応するビジュアルエフェクト
- **Spout2出力対応** — OBS等の配信ソフトに映像を直接送信
- **OSC通信** — 外部コントローラやソフトとリアルタイム連携

---

## できること

| 機能         | 説明                                         |
| ------------ | -------------------------------------------- |
| カメラ操作   | 複数カメラの切り替えと自動カメラワーク       |
| ライト演出   | グループ管理によるライティングオペレーション |
| レーザー演出 | インタラクティブなレーザーエフェクト         |
| トラッキング | MediaPipeによるフルボディトラッキング        |
| アバター表示 | VRMアバターの表示・制御                      |
