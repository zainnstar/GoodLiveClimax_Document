# テレポート (Teleport)

アセットやキャラクターの位置・向きを瞬時に変更するノード群です。

---

## ノード一覧

| 英語名                | 日本語名                     | 説明                                     |
| --------------------- | ---------------------------- | ---------------------------------------- |
| Teleport To Position  | 位置へテレポート             | 指定座標にアセットを即時移動させる       |
| Teleport To Transform | トランスフォームへテレポート | 別オブジェクトの位置・向きにスナップする |
| Teleport Avatar       | アバターをテレポート         | VRM アバターを指定座標に即時移動させる   |
