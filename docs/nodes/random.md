# ランダム (Random)

乱数を生成するノード群です。重みつき抽選・重複なし抽選など高度な乱数制御が可能です。

---

## ノード一覧

| 英語名               | 日本語名          | 説明                                            |
| -------------------- | ----------------- | ----------------------------------------------- |
| Random Int Range     | Int 乱数範囲      | 指定範囲内でランダムな int 値を生成             |
| Random Float Range   | float 乱数範囲    | 指定範囲内でランダムな float 値を生成           |
| Weight Random        | 重み付き乱数      | 重みに基づいて確率的に選択                      |
| No-Repeat Random Int | 重複なし Int 乱数 | 直前と同じ値を繰り返さずにランダムな int を選択 |
