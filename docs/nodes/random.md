# Random ノード

乱数を生成するノード群です。重みつき抽選や重複なし抽選など高度な乱数制御が可能です。

---

## ノード一覧

### Random Int (range)

指定した範囲内のランダムな整数を返します。

| ピン   | 方向 | 型  | 説明           |
| ------ | ---- | --- | -------------- |
| Min    | 入力 | int | 最小値（以上） |
| Max    | 入力 | int | 最大値（未満） |
| Result | 出力 | int | 乱数値         |

### Random Float (range)

指定した範囲内のランダムな浮動小数点数を返します。

| ピン   | 方向 | 型    | 説明   |
| ------ | ---- | ----- | ------ |
| Min    | 入力 | float | 最小値 |
| Max    | 入力 | float | 最大値 |
| Result | 出力 | float | 乱数値 |

### No-Repeat Random Int

直前と同じ値が連続しないランダムな整数を返します。

| ピン   | 方向 | 型     | 説明               |
| ------ | ---- | ------ | ------------------ |
| →      | 入力 | フロー | 次の乱数を生成     |
| Min    | 入力 | int    | 最小値             |
| Max    | 入力 | int    | 最大値             |
| Result | 出力 | int    | 直前と異なる乱数値 |

!!! tip
連番演出や重複なしのアセット選択に使用します。

### Weight Random

重み付きの抽選を行います。重みが高いほど選ばれやすくなります。

| ピン    | 方向 | 型            | 説明                 |
| ------- | ---- | ------------- | -------------------- |
| →       | 入力 | フロー        | 抽選実行             |
| Weights | 入力 | list（float） | 各選択肢の重み       |
| Result  | 出力 | int           | 選ばれたインデックス |

**使用例:** ウェイト `[10, 3, 1]` を設定すると、0が最も選ばれやすく、2が最も選ばれにくくなります。
