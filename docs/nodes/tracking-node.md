# トラッキング (Tracking)

モーションキャプチャ・フェイストラッキングのデータをノードで取得・利用するノード群です。

---

## ノード一覧

| 英語名             | 日本語名             | 説明                                         |
| ------------------ | -------------------- | -------------------------------------------- |
| Get Bone Position  | ボーン位置取得       | 指定ボーンのワールド座標を取得する           |
| Get Bone Rotation  | ボーン回転取得       | 指定ボーンの回転（Quaternion）を取得する     |
| Get Blend Shape    | ブレンドシェイプ取得 | 指定名のブレンドシェイプの現在値を取得する   |
| Set Blend Shape    | ブレンドシェイプ設定 | ブレンドシェイプの値を上書き設定する         |
| Get Face Parameter | 顔パラメータ取得     | フェイストラッキングのパラメータ値を取得する |
| Is Tracking Active | トラッキング有効確認 | トラッキングが有効かどうかを bool で返す     |
