# シーン (Scene)

シーンの読み込み・切り替え・状態管理を行うノード群です。

---

## ノード一覧

| 英語名           | 日本語名             | 説明                                           |
| ---------------- | -------------------- | ---------------------------------------------- |
| Load Scene       | シーンを読み込む     | 指定したシーンを非同期でロードする             |
| Unload Scene     | シーンを解放         | 指定したシーンをアンロードする                 |
| Get Active Scene | アクティブシーン取得 | 現在アクティブなシーン名を返す                 |
| Set Active Scene | アクティブシーン設定 | ロード済みシーンの中から表示シーンを切り替える |
