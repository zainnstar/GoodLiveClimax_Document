# Function ノード

カスタム関数の定義と呼び出しを行うノード群です。BP（ブループリント）間をまたいだ再利用可能な処理を作成できます。

---

## ノード一覧

### Function Definition

関数の「定義」を行うノードです。このノードが関数の実装本体となります。

| 設定項目      | 説明                                                                                                |
| ------------- | --------------------------------------------------------------------------------------------------- |
| Function Name | 関数名（同一BP内でユニークな名前)                                                                   |
| 変数ピン      | 引数として受け取る変数。型は下記から選択: `bool / int / float / string / Vector2 / Vector3 / Color` |

```
→ (入力フロー)
変数1, 変数2... (入力変数ピン)
→ (出力フロー) ← 関数として実行されるノードを繋ぐ
```

---

### Function Call

定義した関数を呼び出すノードです。**別のBPオブジェクトの関数も呼び出すことができます。**

| ピン             | 方向 | 型     | 説明                                                   |
| ---------------- | ---- | ------ | ------------------------------------------------------ |
| →                | 入力 | フロー | 呼び出し実行                                           |
| Target           | 入力 | object | 関数が定義されているBPオブジェクト（省略時は自分自身） |
| Function         | 設定 | -      | 呼び出す関数名                                         |
| 変数ピン（動的） | 入力 | 各型   | Function Definitionに対応した引数                      |
| →                | 出力 | フロー | 関数実行完了後                                         |

!!! tip
Function Call の引数ピンは Function Definition 側の変数定義に合わせて**自動的に追加・変更**されます。

!!! info "BP横断呼び出し"
Target ピンに別のBPオブジェクトの参照を繋ぐことで、そのBPに定義した関数を呼び出せます。これにより複数のBP間で処理を共有・再利用できます。
