# 関数 (Function)

カスタム関数の定義と呼び出しを行うノード群です。BP（ブループリント）間をまたいだ再利用が可能です。

---

## ノード一覧

| 英語名              | 日本語名         | 説明                                                                                              |
| ------------------- | ---------------- | ------------------------------------------------------------------------------------------------- |
| Function Definition | 関数定義         | 関数の本体を定義する。引数の型は bool / int / float / string / Vector2 / Vector3 / Color から選択 |
| Function Call       | 関数呼び出し実行 | 定義した関数を呼び出す。別 BP の関数も呼び出し可能                                                |
