# BP制御 (BP Control)

BP（ブループリント）オブジェクト自体の有効化・無効化・参照取得を行うノード群です。

---

## ノード一覧

| 英語名        | 日本語名            | 説明                                               |
| ------------- | ------------------- | -------------------------------------------------- |
| Enable BP     | BP を有効化         | 指定した BP オブジェクトを有効にする               |
| Disable BP    | BP を無効化         | 指定した BP オブジェクトを無効にする               |
| Get BP Object | BP オブジェクト取得 | 名前またはタグで BP オブジェクトの参照を取得       |
| Get Self      | 自身を取得          | このノードが属する BP オブジェクト自身の参照を返す |
| Get BP Bool   | BP Bool 取得        | 別 BP オブジェクトの Bool 変数を読み取る           |
| Get BP Int    | BP Int 取得         | 別 BP オブジェクトの Int 変数を読み取る            |
| Get BP Float  | BP Float 取得       | 別 BP オブジェクトの Float 変数を読み取る          |
