# Twitch

Twitch チャットおよびチャンネルポイントイベントをトリガーとして利用するノード群です。

---

## ノード一覧

| 英語名            | 日本語名                 | 説明                                             |
| ----------------- | ------------------------ | ------------------------------------------------ |
| On Twitch Chat    | Twitch チャット受信時    | チャットメッセージを受信したとき実行             |
| On Channel Points | チャンネルポイント使用時 | 指定のチャンネルポイント報酬が使用されたとき実行 |
| On Follow         | フォロー時               | 新規フォローが発生したとき実行                   |
| On Subscription   | サブスクリプション時     | サブスクリプションイベント発生時に実行           |
| On Bits           | Bits 受信時              | Bits が送られたとき実行                          |
| Send Twitch Chat  | Twitch チャット送信      | チャットにメッセージを送信する                   |
