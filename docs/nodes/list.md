# List ノード

リスト（複数要素のコレクション）を操作するノード群です。

---

## ノード一覧

### Make List

複数の値からリストを作成します。要素ピンは動的に追加できます。

| ピン              | 方向 | 型   | 説明               |
| ----------------- | ---- | ---- | ------------------ |
| Item 0, Item 1... | 入力 | 任意 | リストに含める要素 |
| Result            | 出力 | list | 生成されたリスト   |

### Get (index)

リストから指定したインデックスの要素を取得します。

| ピン   | 方向 | 型     | 説明                            |
| ------ | ---- | ------ | ------------------------------- |
| List   | 入力 | list   | 対象リスト                      |
| Index  | 入力 | int    | 取得するインデックス（0始まり） |
| Result | 出力 | 要素型 | 取得した要素                    |

### Count

リストの要素数を返します。

| ピン   | 方向 | 型   | 説明       |
| ------ | ---- | ---- | ---------- |
| List   | 入力 | list | 対象リスト |
| Result | 出力 | int  | 要素数     |

### Add

リストに要素を追加します（実行フロー付き）。

| ピン | 方向 | 型     | 説明               |
| ---- | ---- | ------ | ------------------ |
| →    | 入力 | フロー | 実行               |
| List | 入力 | list   | 対象リスト（参照） |
| Item | 入力 | 要素型 | 追加する要素       |
| →    | 出力 | フロー | 追加後             |

### Clear

リストの全要素を削除します。

### For Each

リストの各要素に対して繰り返し処理を行います。

| ピン          | 方向 | 型     | 説明               |
| ------------- | ---- | ------ | ------------------ |
| →             | 入力 | フロー | 実行               |
| List          | 入力 | list   | 対象リスト         |
| → (Loop Body) | 出力 | フロー | 各要素に対して実行 |
| Element       | 出力 | 要素型 | 現在の要素         |
| Index         | 出力 | int    | 現在のインデックス |
| → (Completed) | 出力 | フロー | ループ完了後       |
