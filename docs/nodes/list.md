# リスト (List)

リスト（複数要素のコレクション）を操作するノード群です。

---

## ノード一覧

| 英語名          | 日本語名                   | 説明                                   |
| --------------- | -------------------------- | -------------------------------------- |
| Remove At Index | リストからインデックス削除 | 指定インデックスの要素をリストから削除 |
| Get From List   | リストから取得             | 指定インデックスの要素を取得           |
| Contains        | リストに含む               | リストに指定の要素が含まれるか確認     |
| Set In List     | リストに設定               | 指定インデックスの要素を上書き         |
| Add To List     | リストに追加               | リストの末尾に要素を追加               |
| Append To List  | リストに追記               | 別のリストを末尾に連結                 |
| Clear List      | リストをクリア             | リストの全要素を削除                   |
| Slice List      | リストを切り出し           | 指定範囲の要素だけ取り出す             |
| Get List Count  | リスト数を取得             | リストの要素数を返す                   |
