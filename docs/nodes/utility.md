# ユーティリティ (Utility)

汎用的な補助処理を行うノード群です。

---

## ノード一覧

| 英語名               | 日本語名             | 説明                                   |
| -------------------- | -------------------- | -------------------------------------- |
| Print Screen         | スクリーン印字       | 画面上にデバッグテキストを一時表示する |
| Wait Seconds         | 秒数待機             | 指定秒数だけ処理を待機してから続行する |
| Comment              | コメント             | ノードグラフに注釈コメントを追加する   |
| Reroute              | 中継ノード           | ワイヤーの経路を整理するための中継点   |
| Get Frame Count      | フレームカウント取得 | 経過フレーム数を返す                   |
| Get Time Since Start | 起動からの経過時間   | アプリ起動からの経過時間（秒）を返す   |
