# Asset（アセット）

シーン内のアセット（3D オブジェクト・VRM アバター等）の表示・非表示・参照取得を行うノード群です。

---

## ノード一覧

| 英語名                  | 日本語名           | 説明                                   |
| ----------------------- | ------------------ | -------------------------------------- |
| Show Asset              | アセットを表示     | 指定したアセットを表示状態にする       |
| Hide Asset              | アセットを非表示   | 指定したアセットを非表示状態にする     |
| Toggle Asset Visibility | アセット表示切替   | アセットの表示/非表示を反転する        |
| Get Asset By Name       | 名前でアセット取得 | 名前でシーン内のアセット参照を取得     |
| Spawn Asset             | アセットをスポーン | プレハブからアセットをシーンに生成する |
| Destroy Asset           | アセットを削除     | シーン内のアセットを削除する           |
