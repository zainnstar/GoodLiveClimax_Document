# Flow Control ノード

実行フローの分岐・繰り返し・切り替えを制御するノード群です。

---

## ノード一覧

### Counter

入力フローが来るたびにカウントを1増やし、設定した上限に達したらリセットします。

```
入力フロー（上） → カウントアップ
Reset（下）     → カウントをリセット
```

| ピン      | 方向 | 型     | 説明               |
| --------- | ---- | ------ | ------------------ |
| → (Count) | 入力 | フロー | カウントアップ     |
| → (Reset) | 入力 | フロー | カウントをリセット |
| Count     | 出力 | int    | 現在のカウント値   |
| Max       | 設定 | int    | カウント上限       |

---

### Do N

入力フローをN回だけ通過させます。N回を超えると無効になります。

```
入力フロー（上） → カウント付き通過
Reset（下）     → リセット
```

| ピン      | 方向 | 型     | 説明         |
| --------- | ---- | ------ | ------------ |
| → (Do)    | 入力 | フロー | 実行試行     |
| → (Reset) | 入力 | フロー | リセット     |
| →         | 出力 | フロー | N回まで通過  |
| N         | 設定 | int    | 通過可能回数 |

---

### Flip Flop

入力フローが来るたびにA→B→A→... と交互に出力が切り替わります。

- **A出力**: bool=`true`、int=`0`
- **B出力**: bool=`false`、int=`1`

| ピン    | 方向 | 型     | 説明         |
| ------- | ---- | ------ | ------------ |
| →       | 入力 | フロー | トリガー     |
| → A     | 出力 | フロー | 奇数回目     |
| → B     | 出力 | フロー | 偶数回目     |
| IsA     | 出力 | bool   | A側なら true |
| IndexAB | 出力 | int    | A=0, B=1     |

---

### Sequence

入力フローを複数の出力フローに順番に流します。出力フロー数は可変（追加可能）。

| ピン             | 方向 | 型     | 説明       |
| ---------------- | ---- | ------ | ---------- |
| →                | 入力 | フロー | 実行       |
| → 0, → 1, → 2... | 出力 | フロー | 順番に実行 |

---

### Branch

条件によって実行フローを分岐させます（if文相当）。

| ピン      | 方向 | 型     | 説明                |
| --------- | ---- | ------ | ------------------- |
| →         | 入力 | フロー | 実行                |
| Condition | 入力 | bool   | 条件                |
| → True    | 出力 | フロー | 条件が true の場合  |
| → False   | 出力 | フロー | 条件が false の場合 |

---

### Switch (float / int / string)

入力値によって実行フローを切り替えます。

入力フィールドに比較対象の値を入力し、一致した出力フローが実行されます。

| ピン                | 方向 | 型               | 説明                   |
| ------------------- | ---- | ---------------- | ---------------------- |
| →                   | 入力 | フロー           | 実行                   |
| Value               | 入力 | float/int/string | 比較する値             |
| → [値1], → [値2]... | 出力 | フロー           | 一致した分岐           |
| → Default           | 出力 | フロー           | どれにも一致しない場合 |

!!! tip
Switch は出力フローを増やすことで分岐先を追加できます。
