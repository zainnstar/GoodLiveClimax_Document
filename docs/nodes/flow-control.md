# フロー制御 (Flow Control)

実行フローの分岐・繰り返し・切り替えを制御するノード群です。

---

## ノード一覧

| 英語名        | 日本語名     | 説明                                      |
| ------------- | ------------ | ----------------------------------------- |
| For Loop      | Forループ    | 指定回数だけループして実行                |
| Switch Int    | Int で分岐   | int 値に応じてフローを切り替える          |
| Do N Times    | N回実行      | フローを N 回通過させ、超えたら無効化     |
| Switch Float  | float で分岐 | float 値に応じてフローを切り替える        |
| Counter       | カウンター   | 実行ごとにカウントアップ、上限でリセット  |
| Gate          | ゲート       | 開閉状態によってフローを通過/遷断         |
| Do Once       | 一度だけ実行 | 初回のみ実行、Reset で再有効化            |
| Flip Flop     | 交互実行     | 実行のたびに A→B→A→… と切り替わる         |
| Branch        | 分岐         | bool 条件で True/False に分岐（if文相当） |
| Switch String | 文字列で分岐 | 文字列値に応じてフローを切り替える        |
| Delay Execute | 遅延実行     | 指定秒後にフローを実行                    |
| Sequence      | 順次実行     | 入力フローを複数の出力フローに順番に流す  |
