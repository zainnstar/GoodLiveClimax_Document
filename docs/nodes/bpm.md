# BPM

BPM（テンポ）の設定・取得・同期を行うノード群です。演出と音楽のタイミングを合わせるために使用します。

---

## ノード一覧

| 英語名     | 日本語名       | 説明                                  |
| ---------- | -------------- | ------------------------------------- |
| Set BPM    | BPM を設定     | グローバル BPM 値を設定する           |
| Get BPM    | BPM を取得     | 現在の BPM 値を返す                   |
| BPM Tap    | BPM タップ     | タップイベントから BPM を自動計算する |
| Beat Count | ビートカウント | 現在の拍カウント（ビート番号）を返す  |
