# 時間 (Time)

時間・タイマー・日時の取得・計算を行うノード群です。

---

## ノード一覧

| 英語名          | 日本語名         | 説明                                   |
| --------------- | ---------------- | -------------------------------------- |
| Get Delta Time  | デルタタイム取得 | 前フレームからの経過時間（秒）を返す   |
| Get Time        | 現在時刻取得     | 現在の時刻（時/分/秒）を返す           |
| Get Date        | 現在日付取得     | 現在の日付（年/月/日）を返す           |
| Start Timer     | タイマー開始     | カウントアップタイマーを開始する       |
| Stop Timer      | タイマー停止     | タイマーを停止する                     |
| Reset Timer     | タイマーリセット | タイマーをゼロにリセットする           |
| Get Timer Value | タイマー値取得   | 現在のタイマー経過時間（秒）を返す     |
| Elapsed Time    | 経過時間         | 指定イベントからの経過時間（秒）を返す |
