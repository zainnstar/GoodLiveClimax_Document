# デバッグ (Debug)

演出制作中の動作確認・値のモニタリングに使用するノード群です。

---

## ノード一覧

| 英語名            | 日本語名         | 説明                                        |
| ----------------- | ---------------- | ------------------------------------------- |
| Debug Log         | デバッグログ     | 任意の値を Unity Console に出力する         |
| Debug Log Warning | 警告ログ         | 警告レベルでメッセージを Console に出力する |
| Debug Draw Line   | デバッグ線描画   | シーンビューに一時的なデバッグ線を描画する  |
| Break Point       | ブレークポイント | 実行時に一時停止してデバッグを支援する      |
