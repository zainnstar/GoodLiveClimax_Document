# トランスフォーム (Transform)

アセットの位置・回転・スケールを操作するノード群です。

---

## ノード一覧

| 英語名         | 日本語名       | 説明                                           |
| -------------- | -------------- | ---------------------------------------------- |
| Set Position   | 位置を設定     | アセットのワールド位置を設定する               |
| Set Rotation   | 回転を設定     | アセットのワールド回転（オイラー角）を設定する |
| Set Scale      | スケールを設定 | アセットのローカルスケールを設定する           |
| Get Position   | 位置を取得     | アセットのワールド位置を返す                   |
| Get Rotation   | 回転を取得     | アセットのワールド回転（オイラー角）を返す     |
| Get Scale      | スケールを取得 | アセットのローカルスケールを返す               |
| Move Towards   | 目標へ移動     | 指定速度で目標位置に向かって毎フレーム移動する |
| Rotate Towards | 目標へ回転     | 指定速度で目標回転に向かって毎フレーム回転する |
| Look At        | 注視           | 指定のターゲットに向けてオブジェクトを向かせる |
