# Vector3

3D ベクトル（X/Y/Z）の生成・演算・変換を行うノード群です。

---

## ノード一覧

| 英語名            | 日本語名         | 説明                                  |
| ----------------- | ---------------- | ------------------------------------- |
| Get BP Vector3    | BP Vector3 取得  | BP 変数（Vector3 型）の値を読み取る   |
| Set BP Vector3    | BP Vector3 設定  | BP 変数（Vector3 型）に値を書き込む   |
| Make Vector3      | Vector3 を生成   | X / Y / Z 成分から Vector3 を生成     |
| Break Vector3     | Vector3 を分解   | Vector3 を X / Y / Z に分解して出力   |
| Add Vector3       | Vector3 加算     | 2 つの Vector3 を加算する             |
| Subtract Vector3  | Vector3 減算     | 2 つの Vector3 を減算する             |
| Scale Vector3     | Vector3 スケール | Vector3 を float でスケーリングする   |
| Cross Product     | 外積             | 2 つの Vector3 の外積を返す           |
| Dot Product       | 内積             | 2 つの Vector3 の内積（float）を返す  |
| Lerp Vector3      | Vector3 補間     | 2 つの Vector3 を T（0〜1）で線形補間 |
| Magnitude Vector3 | Vector3 の大きさ | Vector3 の長さ（magnitude）を返す     |
| Normalize Vector3 | Vector3 を正規化 | 長さ 1 の方向ベクトルに変換する       |
