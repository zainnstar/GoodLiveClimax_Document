# カラー (Color)

色の生成・変換・補間を行うノード群です。

---

## ノード一覧

| 英語名       | 日本語名            | 説明                                    |
| ------------ | ------------------- | --------------------------------------- |
| Get BP Color | BP Colorを取得      | BP変数（Color型）の値を読み取る         |
| Set BP Color | BP Colorを設定      | BP変数（Color型）に値を書き込む         |
| HDR Color    | HDR カラー          | HDR対応のカラーリテラル                 |
| HSV to Color | HSV をカラーに変換  | 色相・彩度・明度からカラーを生成        |
| RGB Color    | RGB カラー          | R / G / B の 3 成分からカラーを生成     |
| RGBA Color   | RGBA カラー         | R / G / B / A の 4 成分からカラーを生成 |
| Color to HSV | カラーを HSV に変換 | カラーを H / S / V に分解して出力       |
| Color List   | カラーリスト        | 複数のカラーを 1 つのリストにまとめる   |
| Lerp Color   | カラー補間          | 2 つのカラーを T（0～1）で線形補間      |
