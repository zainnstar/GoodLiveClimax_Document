# Color ノード

色（RGBA）の生成・変換・演算を行うノード群です。

---

## ノード一覧

### Color Literal

固定の色値を出力するノードです。エディタ上のカラーピッカーで色を設定します。

| ピン   | 方向 | 型    | 説明       |
| ------ | ---- | ----- | ---------- |
| Result | 出力 | color | 設定した色 |

### Make Color (RGBA)

R・G・B・A の4成分から色を生成します。

| ピン   | 方向 | 型    | 説明            |
| ------ | ---- | ----- | --------------- |
| R      | 入力 | float | 赤 (0〜1)       |
| G      | 入力 | float | 緑 (0〜1)       |
| B      | 入力 | float | 青 (0〜1)       |
| A      | 入力 | float | アルファ (0〜1) |
| Result | 出力 | color | 生成された色    |

### Make Color (HSV)

色相・彩度・明度から色を生成します。

| ピン   | 方向 | 型    | 説明         |
| ------ | ---- | ----- | ------------ |
| H      | 入力 | float | 色相 (0〜1)  |
| S      | 入力 | float | 彩度 (0〜1)  |
| V      | 入力 | float | 明度 (0〜1)  |
| Result | 出力 | color | 生成された色 |

### Break Color

色をRGBA成分に分解します。

| ピン          | 方向 | 型    | 説明          |
| ------------- | ---- | ----- | ------------- |
| Value         | 入力 | color | 入力色        |
| R / G / B / A | 出力 | float | 各成分 (0〜1) |

### Lerp Color

2つの色を線形補間します。

| ピン   | 方向 | 型    | 説明            |
| ------ | ---- | ----- | --------------- |
| A      | 入力 | color | 開始色          |
| B      | 入力 | color | 終了色          |
| T      | 入力 | float | 補間係数 (0〜1) |
| Result | 出力 | color | 補間された色    |
