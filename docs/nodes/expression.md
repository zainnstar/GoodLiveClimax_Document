# Expression / Math

数式の文字列入力で計算を行うノードです。

---

## ノード一覧

| 英語名          | 日本語名 | 説明                                        |
| --------------- | -------- | ------------------------------------------- |
| Math Expression | 数式計算 | 文字式で数式を入力して float 結果を出力する |

---

## Math Expression 詳細

ノードのフィールドに数式を直接入力します。式内の変数名は自動的に入力ピンとして追加されます。

```
入力式例: a * sin(b) + c
→ a, b, c の入力ピンが自動生成される
```

使用できる演算子:

| 演算子                  | 説明               |
| ----------------------- | ------------------ |
| `+` `-` `*` `/`         | 四則演算           |
| `%`                     | 剰余               |
| `^`                     | べき乗             |
| `sin()` `cos()` `tan()` | 三角関数           |
| `sqrt()` `abs()`        | 平方根・絶対値     |
| `floor()` `ceil()`      | 切り捨て・切り上げ |
| `min()` `max()`         | 最小値・最大値     |
| `pi`                    | 円周率             |
