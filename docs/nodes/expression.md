# Expression ノード

数式・文字列の評価や変換を行うノード群です。

---

## ノード一覧

### Math Expression

文字列で数式を記述して評価します。変数も使用できます。

| ピン             | 方向 | 型     | 説明                            |
| ---------------- | ---- | ------ | ------------------------------- |
| Expression       | 設定 | string | 数式テキスト（例: `a + b * 2`） |
| 変数ピン（動的） | 入力 | float  | 式内の変数                      |
| Result           | 出力 | float  | 計算結果                        |

### String Format

文字列テンプレートに値を埋め込みます。

| ピン          | 方向 | 型     | 説明                            |
| ------------- | ---- | ------ | ------------------------------- |
| Format        | 設定 | string | テンプレート（例: `速度: {0}`） |
| Arg0, Arg1... | 入力 | string | 埋め込む値                      |
| Result        | 出力 | string | 完成した文字列                  |

### String Concat

複数の文字列を連結します。

| ピン   | 方向 | 型     | 説明       |
| ------ | ---- | ------ | ---------- |
| A      | 入力 | string | 前の文字列 |
| B      | 入力 | string | 後の文字列 |
| Result | 出力 | string | 結合結果   |

### Int → String / Float → String

数値を文字列型に変換します。

| ピン   | 方向 | 型        | 説明                 |
| ------ | ---- | --------- | -------------------- |
| Value  | 入力 | int/float | 変換元の値           |
| Result | 出力 | string    | 文字列に変換された値 |
