# Material ノード

アセットのマテリアルパラメータを操作するノード群です。

---

## ノード一覧

### Set Material Color

指定したアセットのマテリアルカラーを変更します。

| ピン     | 方向 | 型     | 説明                                   |
| -------- | ---- | ------ | -------------------------------------- |
| →        | 入力 | フロー | 実行                                   |
| Target   | 入力 | object | 対象アセット                           |
| Property | 設定 | string | マテリアルプロパティ名（例: `_Color`） |
| Color    | 入力 | color  | 設定する色                             |
| →        | 出力 | フロー | 設定後                                 |

### Set Material Float

マテリアルのfloatパラメータを変更します。

| ピン     | 方向 | 型     | 説明                            |
| -------- | ---- | ------ | ------------------------------- |
| →        | 入力 | フロー | 実行                            |
| Target   | 入力 | object | 対象アセット                    |
| Property | 設定 | string | プロパティ名（例: `_Metallic`） |
| Value    | 入力 | float  | 設定する値                      |
| →        | 出力 | フロー | 設定後                          |

### Get Material Color

マテリアルのカラー値を取得します。

| ピン     | 方向 | 型     | 説明         |
| -------- | ---- | ------ | ------------ |
| Target   | 入力 | object | 対象アセット |
| Property | 設定 | string | プロパティ名 |
| Result   | 出力 | color  | 現在のカラー |

### Get Material Float

マテリアルのfloat値を取得します。
