# マテリアル (Material)

アセットのマテリアルパラメータを操作するノード群です。

---

## ノード一覧

| 英語名                  | 日本語名                   | 説明                                     |
| ----------------------- | -------------------------- | ---------------------------------------- |
| Set Material Float      | マテリアルfloatを設定      | 指定プロパティ（float）の値を変更        |
| Toggle Material Keyword | マテリアルキーワードを切替 | シェーダーキーワードの ON/OFF を切り替え |
| Set Material Texture    | マテリアルテクスチャを設定 | テクスチャを差し替える                   |
| Get Material Property   | マテリアルプロパティを取得 | 指定プロパティの現在値を読み取る         |
| Set Material Color      | マテリアル色を設定         | 指定プロパティ（Color）の値を変更        |
