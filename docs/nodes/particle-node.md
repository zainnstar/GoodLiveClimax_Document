# パーティクル (Particle)

パーティクルシステムの再生・停止・パラメータ制御を行うノード群です。

---

## ノード一覧

| 英語名             | 日本語名               | 説明                                     |
| ------------------ | ---------------------- | ---------------------------------------- |
| Play Particle      | パーティクル再生       | 指定のパーティクルシステムを再生する     |
| Stop Particle      | パーティクル停止       | 指定のパーティクルシステムを停止する     |
| Pause Particle     | パーティクル一時停止   | 指定のパーティクルシステムを一時停止する |
| Set Particle Rate  | パーティクルレート設定 | Emission Rate（毎秒の放出数）を変更する  |
| Set Particle Color | パーティクルカラー設定 | パーティクルの開始カラーを設定する       |
