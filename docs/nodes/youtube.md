# YouTube

YouTube Live のチャット・スーパーチャット・メンバーシップイベントを受信するノード群です。

---

## ノード一覧

| 英語名           | 日本語名                 | 説明                                 |
| ---------------- | ------------------------ | ------------------------------------ |
| On YouTube Chat  | YouTube チャット受信時   | チャットメッセージを受信したとき実行 |
| On Super Chat    | スーパーチャット受信時   | スーパーチャットが送られたとき実行   |
| On Membership    | メンバーシップ受信時     | メンバーシップイベント発生時に実行   |
| On Super Sticker | スーパーステッカー受信時 | スーパーステッカーを受信したとき実行 |
