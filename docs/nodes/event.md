# Event ノード

システムイベントや外部トリガーをBP（ブループリント）の実行フローに接続するノード群です。

---

## ノード一覧

### On Start

BPオブジェクトがシーンに読み込まれた直後に1回だけ実行されます。

| ピン | 方向 | 型     | 説明         |
| ---- | ---- | ------ | ------------ |
| →    | 出力 | フロー | 開始時に発火 |

### On Update

毎フレーム実行されるイベントです。

| ピン      | 方向 | 型     | 説明                           |
| --------- | ---- | ------ | ------------------------------ |
| →         | 出力 | フロー | 毎フレーム発火                 |
| DeltaTime | 出力 | float  | 前フレームからの経過時間（秒） |

### On OSC Receive

OSCメッセージを受信したときに発火するイベントです。

| ピン    | 方向 | 型     | 説明                |
| ------- | ---- | ------ | ------------------- |
| Address | 設定 | string | 受信するOSCアドレス |
| →       | 出力 | フロー | 受信時に発火        |
| Values  | 出力 | list   | 受信したパラメータ  |

### On BPM Beat

BPM同期イベント。指定した拍数ごとに発火します。

| ピン | 方向 | 型     | 説明                 |
| ---- | ---- | ------ | -------------------- |
| Beat | 設定 | int    | 何拍ごとに発火するか |
| →    | 出力 | フロー | 指定拍ごとに発火     |
