# ノードの概要

GoodLiveClimaxには、**ノードエディタ**が組み込まれています。  
ビジュアルプログラミングのようにノードを繋いで、演出の自動化・条件分岐・パラメータ連動などを構築できます。

---

## ノードエディタとは

- マウスのドラッグ＆ドロップでノードを繋ぐだけで、複雑なロジックを組める
- プログラミングの知識不要
- ライブ中にリアルタイムで演出をコントロールする「自動化レシピ」を作れる

---

## 基本操作

| 操作 | 方法 |
|------|------|
| ノードの追加 | 右クリック → カテゴリから選択 |
| ノードの削除 | ノードを選択して Delete キー |
| ノードの接続 | ピンをドラッグして別のピンに繋ぐ |
| 接続の切断 | 接続線を右クリック → 削除 |
| 複数選択 | Ctrl+クリック / 範囲ドラッグ |

---

## ピンの種類

| ピン | 色 | 説明 |
|------|-----|------|
| Flow（実行フロー） | 白 | ノードの実行順を繋ぐ |
| bool | 黄 | true / false |
| int | 青緑 | 整数値 |
| float | 緑 | 小数値 |
| string | ピンク | 文字列 |
| color | 橙 | RGBA色情報 |
| vector2 / vector3 | 水色 | 2次元・3次元座標 |

---

## BP（ブループリント）

ノードグラフは**BP（ブループリント）**単位で保存・切り替えができます。  
曲ごと・シーンごとに複数のBPを用意しておくことができます。

関数ノードはBPをまたいで呼び出すことができます。
